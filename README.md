# TuLens

## Requirements
- Project Setup
    - Node.js
- Usage
    - Shibir card

## Application Setup
1. Install the dependencies: ```npm install```
2. Run the application using npx: ```npx http-server . -p <PORT> -c-1```

## Usage
1. Run the application
2. Scan the shibir card
3. Point it a human's face to reveal Guruhari's murti
